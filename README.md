# MonogusaAddonStore
ヴァカリネ鯖のものぐさ12が作ったアドオン置き場。  
  
indunplus キャラクター毎のインスタンスダンジョンや防衛戦の回数を表示する  
/idpで表示非表示入り替え  
![indunplus](http://i.imgur.com/mAWAEWB.jpg "こんな感じ")

  
MutekiAttitude アウシュリネ女神像のバフ、セーフティーゾーンのバフ、無敵の草のやつを強調表示  
/muteki on/offで有効無効切り替え
