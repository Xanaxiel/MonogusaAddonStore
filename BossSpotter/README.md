# Boss Spotter
This add-on will automatically report for party or guild where it was when you found the boss.  

/bss on/off  
Boss Spotter enable or disable  
/bss party on/off  
inform for party on/off  
/bss guild on/off  
inform for guild on/off  
/bss lang japanese  
inform by japanese  
/bss lang english  
inform by english
